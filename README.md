# Econometrics in Python
## Эконометрические исследования с использованием Python
Исследование изучаемого явления или объекта состоит из нескольких этапов:
 1. формирование цели исследования;
 2. сбор данных (в результате наблюдения или измерения);
 3. предварительная обработка (очистка, удаление пропусков и повторов); 
 4. изучение статистических свойств (описательная статистика и согласование с видом распределения);
 5. формирование и проверка статистических гипотез;
 6. построение моделей, их статистическое обоснование и проверка точности предсказания.
